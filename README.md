# SermoServer

Backend of Sermo, a word game.

This is part of the final project of course: Object-Oriented Programming Using C++.

## Requirements

- libconfig++
- libsqlite3
- g3logger

## Build

### Linux/macOS

```bash
$ git clone https://github.com/imtsuki/SermoServer.git
$ cd SermoServer/
$ make
$ ./build/server
```
